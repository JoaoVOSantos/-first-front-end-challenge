Inter Variable Font
===================

This download contains Inter as both a variable font and static fonts.
Este download contém Inter como fonte variável e fontes estáticas.


Inter is a variable font with these axes:
Inter é uma fonte variável com estes eixos:
  slnt
  wght

This means all the styles are contained in a single file:
Isso significa que todos os estilos estão contidos em um único arquivo:

  Inter-VariableFont_slnt,wght.ttf

If your app fully supports variable fonts, you can now pick intermediate styles that aren’t available as static fonts. Not all apps support variable fonts, and in those cases you can use the static font files for Inter:

Se o seu aplicativo oferece suporte total a fontes variáveis, agora você pode escolher estilos intermediários que não estão disponíveis como fontes estáticas. Nem todos os aplicativos suportam fontes variáveis ​​e, nesses casos, você pode usar os arquivos de fontes estáticas para o Inter:
  static/Inter-Thin.ttf
  static/Inter-ExtraLight.ttf
  static/Inter-Light.ttf
  static/Inter-Regular.ttf
  static/Inter-Medium.ttf
  static/Inter-SemiBold.ttf
  static/Inter-Bold.ttf
  static/Inter-ExtraBold.ttf
  static/Inter-Black.ttf

Get started
Vamos começar
-----------

1. Install the font files you want to use
1. Instale os arquivos de fonte que deseja usar

2. Use your app's font picker to view the font family and all the
available styles
2. Use o seletor de fontes do seu aplicativo para visualizar a família de fontes e todas os estilos disponíveis
